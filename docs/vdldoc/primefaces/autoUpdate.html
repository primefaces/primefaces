Tag autoUpdate

  • Description:
    AutoUpdate is a TagHandler to mark the parent component to be updated at every AJAX request. AutoUpdate is ignored if the trigger (p:commandButton, p:ajax...) of AJAX request has the attribute ignoreAutoUpdate set to 'true'.
