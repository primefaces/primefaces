PfAjaxCompleteCallback

PfAjaxCompleteCallback: (this: Document, xhr: jqXHR, settings: AjaxSettings) => void

Callback for when an AJAX request completes, either successfully or with an error. Usually set via <p:ajaxStatus oncomplete="..."/>. This callback applies when <p:ajax /> is used.
