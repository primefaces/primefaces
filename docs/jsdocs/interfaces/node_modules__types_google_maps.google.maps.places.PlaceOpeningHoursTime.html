Interface PlaceOpeningHoursTime

Defines when a Place opens or closes.

Properties

day

day: number

The days of the week, as a number in the range [0, 6], starting on Sunday. For example, 2 means Tuesday.

hours

hours: number

The hours of the google.maps.places.PlaceOpeningHoursTime.time as a number, in the range [0, 23]. This will be reported in the Place’s time zone.

minutes

minutes: number

The minutes of the google.maps.places.PlaceOpeningHoursTime.time as a number, in the range [0, 59]. This will be reported in the Place’s time zone.

Optional nextDate

nextDate?: number

The timestamp (as milliseconds since the epoch, suitable for use with new Date()) representing the next occurrence of this PlaceOpeningHoursTime. It is calculated from the google.maps.places.PlaceOpeningHoursTime.day of week, the google.maps.places.PlaceOpeningHoursTime.time, and the google.maps.places.PlaceResult.utc_offset_minutes. If the google.maps.places.PlaceResult.utc_offset_minutes is undefined, then nextDate will be undefined.

time

time: string

The time of day in 24-hour "hhmm" format. Values are in the range ["0000", "2359"]. The time will be reported in the Place’s time zone.
