Optional center

center: LatLng | LatLngLiteral

Overrides CircleOptions.center

The center of the Circle.

Optional clickable

clickable: boolean

Inherited from CircleOptions.clickable

Indicates whether this Circle handles mouse events. Defaults to true.

Optional draggable

draggable: boolean

Inherited from CircleOptions.draggable

If set to true, the user can drag this circle over the map. Defaults to false.

Optional editable

editable: boolean

Inherited from CircleOptions.editable

If set to true, the user can edit this circle by dragging the control points shown at the center and around the circumference of the circle. Defaults to false.
