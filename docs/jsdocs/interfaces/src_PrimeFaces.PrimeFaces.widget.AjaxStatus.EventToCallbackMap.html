Interface EventToCallbackMap

Maps between the AjaxStatusEventType and the corresponding event handlers. Used by the AjaxStatus component.

Properties

complete

complete: PfAjaxCompleteCallback | OnEventCallback | OnErrorCallback

Callback for when an AJAX request completes, either successfully or with an error. Usually set via <p:ajaxStatus oncomplete="..."/>.

error

error: PfAjaxErrorCallback | OnErrorCallback

Callback for when an AJAX request fails. Usually set via <p:ajaxStatus onerror="..."/>.

start

start: PfAjaxStartCallback | OnEventCallback

Callback for when an AJAX request starts. Usually set via <p:ajaxStatus onstart="..."/>.

success

success: PfAjaxSuccessCallback | OnEventCallback

Callback for when an AJAX request succeeds. Usually set via <p:ajaxStatus onsuccess="..."/>.
