Interface DialogHandlerCfgOptions

Interface of the dialog configuration object for a dialog of the dialog framework. Used by PrimeFaces.dialog.DialogHandlerCfg. This is mainly just the PrimeFaces.widget.DialogCfg, but adds a few more properties.
