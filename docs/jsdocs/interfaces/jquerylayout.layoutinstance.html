resizeAll

  • resizeAll(): void

  • Resizes the entire layout to fit its container. This method runs automatically for all layouts when the browser window is resized. When a layout is inside another element (its container), and that container element can be sized without resizing the entire window, then layout.resizeAll() should be called so the inner layout will resize when its container resizes.

    Returns void

resizeContent

  • resizeContent(pane: BorderPane | CenterPane): void

  • Resizes the content container inside the specified pane. Content is resized automatically when the pane is resized or opened, so this method only needs to be called manually if something changes the height of a header or footer element in the pane.

    Parameters

    • pane: BorderPane | CenterPane

      A pane with content to resize.

    Returns void

show

  • show(pane: BorderPane, openPane?: boolean): void

  • Un-hide the specified pane. Normally also opens the pane, but if you pass false as the second parameter, then pane will un-hide, but be closed (slider-bar and toggler now visible).

    Parameters

    • pane: BorderPane

      A pane to shown.

    • Optional openPane: boolean

      Whether the pane is also opened.

    Returns void

sizePane

  • sizePane(pane: BorderPane, sizeInPixels: number): void

  • This sizes the pane in the direction it opens and closes - for north and south panes, size=outerHeight, for east and west panes, size=outerWidth. The pane will only resize within its minSize and maxSize limits.

    Parameters

    • pane: BorderPane

      A pane to resize.

    • sizeInPixels: number

      The new size in pixels for the pane.

    Returns void

toggle

  • toggle(pane: BorderPane): void

    Parameters

    Returns void
