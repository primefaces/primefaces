# PrimeFaces Theming with SASS

PrimeFaces SASS Themes for JSF/Faces

## Usage

To compile the CSS once:

```shell
mvn clean package
```

or manually:
```shell
mvn dart-sass:compile-sass
```

To watch the SASS files for changes and re-compile:

```shell
mvn dart-sass:watch-sass
```
